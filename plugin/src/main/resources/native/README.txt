This directory contains SQLite native libraries for various platforms.
They will be extracted to the plugin's 'libs' directory at runtime.
The libraries are named according to the platform they support:

- sqlite-native-win-x64.dll: Windows 64-bit
- sqlite-native-win-x86.dll: Windows 32-bit
- libsqlite-native-mac.dylib: macOS
- libsqlite-native-linux-x64.so: Linux 64-bit
- libsqlite-native-linux-x86.so: Linux 32-bit
- libsqlite-native-linux-arm.so: Linux ARM 32-bit
- libsqlite-native-linux-aarch64.so: Linux ARM 64-bit
