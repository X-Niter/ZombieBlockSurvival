===== SEVENTODIE SOUNDS FOLDER =====

Place custom sound files in this folder. The plugin will register these sounds with Minecraft on startup.

Expected folder structure:
- ambient/ - Ambient sounds
- blocks/ - Block-related sounds
- mobs/ - Mob sounds
- ui/ - UI sounds

Sound files should be in .ogg format and named according to the Minecraft naming convention:
[category].[name].ogg

Examples:
- ambient.bloodmoon.ogg
- blocks.upgrade.ogg
- mobs.feral_zombie.ogg
- ui.quest_complete.ogg

The plugin will automatically register these sounds in the plugin.yml file.
